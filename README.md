# PHP EventManager

## installation

#### composer.json
```json

{
    "require": {
        /* ... */
        "statflo/php-event": "dev-master"
    },
    "repositories": [
        /* ... */
        {
            "type": "vcs",
            "no-api": true,
            "url":  "git@github.com:statflo/php-event.git"
        }
    ]
}
```
